# ❌ Tic-Tac-Toe

A browser based Tic-Tac-Toe game built with JavaScript as part of the intermediate JavaScript curriculum from The Odin Project.

The primary goal of this project was to apply the module pattern, factory functions, and object-orientated design principle to create a maintainable application with minimal global scope.

# 🔗 Live Preview

Live Demo: https://kiesheah.github.io/odin-tic-tac-toe/
Repository: https://github.com/KiesheaH/odin-tic-tac-toe

# 🎯 Project Objective

This project focused on practicing:

- Factory functions
- The Module Pattern (IIFEs)
- Encapsulation and private State
- Separating game logic from UI rendering
- DOM manipulation
- Event handling
- Game state management

# 🛠️ Built With

- HTML
- CSS3
- JavaScript (ES6+)
- Factory Functions
- IIFE Module Pattern

# ⚙️ Features

- Two player Tic-Tac-Toe gameplay
- Custom player names
- Win and tie detection
- Restart game functionality
- Prevents moves on occupied cells
- Dynamic board rendering
- Game over status display
